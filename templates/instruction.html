Instructions

To upload, press the Choose File button to begin choosing the file. After choosing the file, press Submit the file to begin checking for grammar, saving the file and downloading the finished product. The user can only submit one file at a time. The user can check their original file by choosing the Check Upload File button, the user can proceed to choose which file they want to download or delete.

Some extra guide on using the program.
- endnotes:
Step 1: (if you are not the writer or do not possess the citation) click “export to endnote” then “export travelling library” in endnote section.
Step 2: Click “convert citations and bibliography” then “Convert to unformatted citations”.
Step 3: submit the file to the website and download the updated file.
Step 4: Open the updated file and click “convert citations and bibliography” in endnote section.

- Skip: Our program can attempt to skip a sentence in a paragraph of a cell in a table when you put them in underline.

- Stop at reference: if you do not write your reference using endnote our program will stop scanning and fixing when it meets the standalone word “reference” or “references”, it only works if those word are alone and not within a normal paragraph.
Ex: This paragraph with the word reference will be scanned and fixed.
  Reference
  This paragraph will not be scanned and fixed since it is below a standalone word reference.

Limitation on grammar fixing:
-Our Grammar fix function has issues fixing text that is too short or does not end with a period or end with a colon ex: “1”, “This is the information:”, “introduction:”, “java”, “hello ”,...
-We can keep most formats of text when replaced, however, if a paragraph has many formats, we can only keep the first one and apply it to the entire paragraph.
-For the skip feature, while skip works perfectly in tables, it may bug out when used in a paragraph sentence. It has to be used in a full sentence and that sentence must all belong in a run element (it must be typed out in one go).
